This folder contains a few examples of real and fake samples in their respective folders.

The sub-folders are named based on the types of nuclei present in the patches inside it.

The nuclei types are listed in the following order,

Background, Neutrophil, Epithelial, Lymphocyte, Plasma, Eosinophil, Connective.

Note that the first number indicating the background is always 1.

For instance, 1_0_0_1_1_1_1 indicates that the patches inside the folder contain Lymphocyte, Plasma, Eosinophil, Connective Nuclei types.
